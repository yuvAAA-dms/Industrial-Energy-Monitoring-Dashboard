# 📌 Project Overview

This project was developed during my internship at **Bharat Heavy Electricals Limited (BHEL), Haridwar**.

The objective was to analyze industrial energy consumption data and design an interactive Power BI dashboard that enables monitoring of:

- Energy Consumption
- Operational Cost
- Machine Performance
- Department-wise Analysis
- Power Factor Monitoring
- Voltage Analysis
- Shift-wise Energy Usage

The dashboard transforms raw industrial data into actionable insights for better operational decision-making.

# 📈 Key Insights

- CNC department consumes the highest amount of energy.
- Critical machines contribute significantly to overall consumption.
- Morning shifts account for the largest share of energy usage.
- Energy costs vary considerably across departments.
- Monitoring power factor helps identify inefficient machine operation.
- Voltage trends remain relatively stable across machines.
